# JustFog
> A mod for Gorilla Tag that forces fog into maps

---

# Controls
- Toggle GUI: Insert

---

# Installation

- Download the release from [here](https://github.com/iiDk-the-actual/JustFog/releases/latest)
- Drag the file `JustFog.dll` to your plugins folder
- Start Gorilla Tag if needed
